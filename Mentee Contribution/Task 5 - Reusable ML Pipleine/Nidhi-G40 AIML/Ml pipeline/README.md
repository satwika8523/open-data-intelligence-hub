Titanic Survival Prediction Using Reusable ML Pipeline

Objective:
To build a reusable machine learning pipeline using scikit-learn.

Tools Used:
- Python
- Pandas
- Scikit-learn
- Matplotlib

Model:
Random Forest Classifier

Pipeline Components:
- SimpleImputer
- StandardScaler
- OneHotEncoder
- ColumnTransformer
- Pipeline

Outcome:
The model predicts whether a passenger survived or not.
